3 Simple Steps

Install

Totally plug-in-play

{% highlight bash %} # Add the gem to your Gemfile $ bundle add rails # Then run the Rails generator $ rails g graphql:install {% endhighlight %}

Define

Designed for simple and complex Schemas

{% highlight ruby %} # app/graphql/app_schema.rb class GraphQL::AppSchema < GraphQL::Schema field(:welcome).resolve { 'Hello World!' } end {% endhighlight %}

Run

Follows Rails core principles

{% highlight bash %} $ curl -d '{"query":"{ welcome }"}' \ -H "Content-Type: application/json" \ -X POST http://localhost:3000/graphql # {"data":{"welcome":"Hello World!"}} {% endhighlight %}
